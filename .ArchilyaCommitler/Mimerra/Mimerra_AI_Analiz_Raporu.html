📋 Yürütülen Görev

Kullanıcı, Mimerra yapay zeka araçlarında promptların yazılıp yazılmadığını kontrol etmemi istedi. Bu rapor, Mimerra'nın AI prompt sisteminin detaylı analizini, tersine mühendislik (reverse engineering) bulgularını ve canlı site üzerinden yakalanan network trafiğini içermektedir.

🎯 Ana Bulgu: Mimerra Prompt Yazıyor mu?

EVET. Mimerra, kullanıcıdan aldığı basit form verilerini (oda tipi, boyutlar, malzemeler) alıp arka planda detaylı Türkçe promptlar oluşturuyor. Bu promptlar, Mimerra'nın kendi backend sunucusunda (mimerra.com/api/render) şablonlarla birleştirilip Google Gemini API'ye gönderiliyor.

Prompt Akış Şeması

Kullanıcı Formu (React Frontend)
Oda tipi, boyutlar, malzemeler, mobilya, stil seçimi
POST /api/render
JSON payload olarak backend'e iletilir
Mimerra Backend
Form verileri + Türkçe system prompt şablonu birleştirilir
Google Gemini API
generativelanguage.googleapis.com/v1beta
Response
AI üretimi metin + görsel (TEXT + IMAGE modality)

materialsMeşe parke, antrasit duvarlar...Yüzey kaplama ve doku talimatları

Önemli Not: contents.parts dizisi boş gönderiliyor. Bu, asıl promptun backend tarafında dinamik olarak oluşturulduğunu kanıtlıyor. Frontend sadece yapılandırılmış veriyi (details objesi) gönderiyor; backend bu veriyi alıp Gemini için detaylı, bağlamsal bir prompt metni oluşturuyor.

🧠 Prompt Mühendisliği Teknikleri

Mimerra'nın backend'inin muhtemelen kullandığı prompt mühendisliği stratejileri:

1. System Instruction (Rol Atama)

"Sen bir iç mimarlık ve görselleştirme uzmanısın. 
Görevin, kullanıcının verdiği detaylara sadık kalarak 
fotogerçekçi mimari render görüntüleri üretmektir."

2. Context Injection (Bağlam Enjeksiyonu)

"Oda Tipi: Yatak Odası
Boyutlar: 4m x 5m, Tavan yüksekliği 2.8m
Malzemeler: Meşe parke, antrasit duvarlar, pirinç detaylar
Mobilya: Geniş L koltuk, mermer orta sehpa, büyük soyut tablo
Stil: Modern"

3. Negative Prompting (Negatif Yönlendirme)

Muhtemelen kullanılan kısıtlamalar:

  • "Gerçek dışı oranlardan kaçın"
  • "Müşteri tarafından talep edilmeyen objeler ekleme"
  • "Bozuk perspektif ve deforme mobilyalar üretme"

4. Few-Shot Prompting (Örnek Tabanlı)

Hazır şablonlar ("Ultra lüks modern salon", "Yemek odası", "Lüks yatak odası") seçildiğinde, backend bu şablonlara ait önceden test edilmiş prompt parametrelerini kullanıyor olabilir.

✅ Sonuç

Mimerra, yapay zeka destekli mimari görselleştirme platformunda aktif olarak prompt mühendisliği yapmaktadır. Kullanıcıdan alınan basit form verileri, Mimerra'nın backend sunucusunda dinamik olarak zengin, bağlamsal ve Türkçe promptlara dönüştürülüp Google Gemini API'ye iletilmektedir.

Prompt şablonlarının tam metni frontend kodunda görünmemektedir; bunlar Mimerra'nın backend altyapısında (muhtemelen Firebase Functions veya özel bir API sunucusu üzerinde) saklanmaktadır.
